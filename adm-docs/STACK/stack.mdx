
## Docker

- https://github.com/mathinstitut/goemaxima#using-the-docker-image

### Using a bundled stackmaxima

- https://hub.docker.com/r/mathinstitut/goemaxima

```bash
# Create a directory for goemaxima
sudo mkdir -p /opt/goemaxima
cd /opt/goemaxima

# Save the compose file (copy from the artifact above)
sudo nano docker-compose.yml

# Create and start goemaxima
sudo docker compose up -d

sudo docker compose pull
```

### Update

```bash

# Create and start goemaxima
sudo docker compose down
sudo docker compose pull
sudo docker compose up -d
```

```bash
docker pull mathinstitut/goemaxima:2026042200-latest
```

docker-compose.yml

```yml
services:
  goemaxima:
    image: mathinstitut/goemaxima:2026042200-latest
    container_name: goemaxima_container
    ports:
      - "8081:8080"
    restart: unless-stopped
    environment:
      - MAXIMA_OPTION=default # replace or extend with actual options if needed
    volumes:
      - ./data:/app/data # mount a volume if the container needs local storage
```
